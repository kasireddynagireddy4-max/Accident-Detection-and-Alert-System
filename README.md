# Accident-Detection-and-Alert-System
An Accident Detection and Alert System is a smart safety system that automatically detects a vehicle accident and sends an emergency alert to predefined contacts or emergency services with the exact location of the accident.  It helps provide fast medical assistance and reduces death rate due to delayed help.

Web Scraping Tool | Python, BeautifulSoup, Requests, Pandas
– Developed a tool to extract data from multiple e-commerce websites by parsing HTML and navigating complex page structures.
– Stored extracted information in structured formats for subsequent data analysis and price comparison.
E-commerce Database Management System | SQL, MySQL
– Managed relational tables including Products, Customers, Orders, and Payments.
– Implemented advanced SQL queries like JOINs to retrieve critical data on inventory and sales trends.
To-Do List Application | Python, File Handling
– Designed a command-line application allowing users to add, view, edit, and delete tasks.
– Utilized file handling for data persistence, ensuring tasks remain saved after application closure.
